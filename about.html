Privacy

Here at Let Me Google That For You we only collect the following information:
  • Whether IPv4 or IPv6 was used. (We do *NOT* store the IP address. Access via IPv4 is stored as "127.0.0.2" and access via IPv6 is stored as "::2" in the nginx access log.)
  • Time stamp of access.
  • The resource accessed from this site.
  • The status code of the request to this site.
  • The size of the request sent by the server.
  • The referer.
